# sizing-markets

## What it does
Estimates TAM, SAM, and SOM for any market opportunity using top-down and bottom-up methodologies, producing an investor-ready PDF report with sourced figures, labelled assumptions, growth projections, and a curated sources list. Best used when sizing a market for due diligence, thesis validation, or an investor pitch.

## Reference files
None.

## Install
Copy this folder into your skills directory, then toggle the skill on in **Customize > Skills**.

## What to change
- Output path in SKILL.md (`Resources/Output files/`) — update to your preferred output folder if different
- Source tier list in Step 2 — add or remove analyst providers to match your preferred research stack

## Usage
**Trigger:** "Size this market for me" / "What's the TAM for X?" / "Build a TAM/SAM/SOM for this opportunity"
**Output:** PDF report saved to your output folder, including market definition, TAM/SAM/SOM with dual methodology, summary table, growth drivers, key assumptions with confidence levels, and a top-20 sources list
