# synthesising-research

## What it does
Condenses batches of research reading — reports, articles, clippings, call transcripts, and DD notes — into a structured insight memo covering key signals, contrarian angles, thesis implications, and action items. Applicable across any research context: thesis development, deal evaluation, or general topic research. Output is intentionally rough — designed for knowledge-building, not stakeholder delivery. Best used after accumulating a batch of sources that need synthesis before further analysis or action.

## Reference files
- `references/synthesis-structure.md` — memo structure template; always read before writing
- `references/source-citation-format.md` — inline citation format; always read before writing
- `references/signal-vs-noise-heuristics.md` — signal filtering heuristics; read if present (built-in defaults apply if missing)

## Install
Copy this folder into your skills directory, then toggle the skill on in **Customize > Skills**.

## What to change
- `references/synthesis-structure.md` — update to reflect your preferred memo structure and section order
- `references/source-citation-format.md` — update to match your citation style
- `references/signal-vs-noise-heuristics.md` — customise to tune signal filtering to your investment criteria

## Usage
**Trigger:** "synthesise this reading"
**Output:** Markdown memo saved to [your output folder]
