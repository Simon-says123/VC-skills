# Synthesis Memo Template

This is the canonical memo structure for synthesising-research. Follow it exactly. All sections are mandatory except Insight Tweets (prompted separately).

---

## Template

```
---
# [SECTOR / THESIS FRAME] | Research Synthesis
**Date:** [Date synthesised]
**Sources:** [N sources confirmed] | **Period covered:** [Date range or 'mixed']
**Thesis frame:** [Stated frame, or 'General scan — no frame applied']
---

## So What
[3 sentences maximum. Write this last. Lead with the single most important investment
implication from this reading. Sentence 2: the key tension or risk. Sentence 3: what
this means for next action. Do not hedge. Do not context-set. If you cannot write this
without hedging, the synthesis has not produced a clear view — say so.]

---

## Key Signals
[3–7 signals only. Each: 1–2 sentences. Inline citation mandatory on every signal.
Format: Signal statement. [Source, Date]. If a signal is single-source or weakly evidenced,
mark it: ⚠️ Single source.

If more than 7 signals appear necessary, the source set is too broad or the signal filter
is too loose. Tighten both before adding signals.]

---

## What's New
[What has genuinely shifted since the last synthesis or review? This section is comparative.
If no prior reference point exists, note that explicitly.

Three sub-buckets (use only those that apply):
- **New facts:** [Specific new data points, names, numbers]
- **Acceleration:** [Known trends that are moving faster than expected — with evidence]
- **Reversals / surprises:** [Things that contradicted the prior view — the most valuable sub-bucket]

If nothing is new and the reading merely confirmed the known, state: "No new signals.
This reading confirms [X] but does not update the thesis." That is a legitimate and useful output.]

---

## Contrarian / Under-Priced Angles
[What is the consensus getting wrong, underweighting, or not yet pricing? This section
justifies the memo's existence. If it is empty, the reading was not novel enough to warrant
synthesis — say so directly rather than padding.

Format: "[Consensus view] — but [what is actually true or emerging] because [evidence]."

Push hard here. Weak contrarian angles are worse than none: they give false confidence.]

---

## Thesis Implications
[Direct implications for live deal work or the fund's investment thesis. Be specific.
Name the deal or thesis strand if relevant. Avoid generic implications ("this is a growing
market") — those belong in Key Signals.

Three sub-buckets (use only those that apply):
- **Strengthens:** [What this reading reinforces about a current position]
- **Complicates:** [What introduces new uncertainty or nuance]
- **Weakens:** [What challenges a current holding or thesis — do not omit this]

If no connection to live work exists, state that. Do not manufacture relevance.]

---

## Action Items
[Concrete next steps only. Format each as: [Owner if known] — [Specific action] — [Timeframe]

Acceptable: "Simon — add parametric re-insurance angle to Company X DD — this week"
Not acceptable: "Monitor the space", "Explore further", "Consider implications"

If no clear action emerges from this synthesis, state that. It is useful data — it means
the reading was either confirmatory or too early-stage to act on.]

---

## Insight Tweets (optional — add only if Simon confirms at Gate 3)
[1–3 tweetable insights. Each under 140 characters. No jargon. No context assumed.
These are for internal sharing, not public posting. They should be standalone.]
```

---

## Section-Level Guidance

**So What**: Draft it last. Read the whole memo before writing it. It should be so clear that someone who reads only this section knows whether they need to read on. If your So What runs to more than 3 sentences, you have not yet found the real view.

**Key Signals**: Ruthless curation only. More than 7 = you haven't curated. A signal without a source citation does not go in. Single-source signals are flagged, not excluded.

**What's New**: Do not re-state Key Signals here. This section requires a reference point. If none exists, note it — do not invent one.

**Contrarian / Under-Priced Angles**: The highest-value section for investment purposes. Empty or weak = low-value synthesis. If you genuinely cannot find a contrarian angle, say "No contrarian angle identified in this reading" — that is honest and useful.

**Thesis Implications**: Must connect to live work. Generic relevance claims belong in Key Signals. Be willing to write under "Weakens" — that takes more courage and is often more valuable.

**Action Items**: If an insight does not produce at least a tentative action, question whether it belongs in the memo. Vague actions are worse than no actions.
