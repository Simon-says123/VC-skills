# Source Citation Format

Use inline citations throughout the memo. No footnotes.

---

## Format

`[Source Name, Date if known]`

Every signal in the Key Signals section must carry a citation. In other sections, cite
claims that are non-obvious or drawn from a specific source. Do not cite general knowledge.

---

## Examples by Source Type

| Source type | Example citation |
|---|---|
| Published report | `[McKinsey Insurtech Report, Mar 2026]` |
| VC / analyst memo | `[Accel State of Fintech Q1 2026]` |
| Expert call | `[Expert call: ex-Zurich CTO, Feb 2026]` |
| DD notes | `[DD notes: Company X, Mar 2026]` |
| News article (named outlet) | `[FT, 14 Apr 2026]` |
| News article (pasted link, no name) | `[bloomberg.com, Apr 2026]` |
| Clipping (undated) | `[Clipping: FT article, undated]` |
| Internal document | `[IAG internal: [doc name], [date]]` |
| Academic paper | `[Author et al., Journal, Year]` |

---

## Rules

1. Every signal in Key Signals must have a citation. No exceptions.
2. If the source date is unknown, write `undated` — do not omit the field.
3. If the same claim appears across multiple sources, cite the most authoritative one
   and note `confirmed across [N] sources` where that corroboration is meaningful.
4. Never fabricate source names or dates. If uncertain: `[Unverified]`.
5. For sources with no identifiable publication name, use the domain or file name.
6. Do not cite sources that were scanned and found to contain no signal — only sources
   that contributed content to the memo require citation.

---

## Source Authority Hierarchy

Use this when sources conflict. Cite both; surface the disagreement; do not silently
resolve by picking the higher-ranked source.

1. Primary research — expert calls, DD notes, Simon's direct observations
2. Proprietary data — paid reports, licensed databases, broker data
3. Quality journalism — FT, Economist, Bloomberg, Reuters
4. Academic and policy — peer-reviewed papers, reputable think-tanks
5. VC and analyst commentary — a16z memos, Sequoia essays, CB Insights
6. Trade press and specialist blogs
7. Company-owned content — press releases, blog posts, investor updates

When two sources of different authority conflict, format the conflict as:
`[Higher-authority source] states X [citation]. [Lower-authority source] contradicts
this, stating Y [citation]. The discrepancy has not been resolved.`

Do not present the higher-authority source as correct by default — authority reduces
but does not eliminate uncertainty.
